1. ERKENNUNG → "Das ist mein Problem." → Hero
2. EINORDNUNG → "Das ist berechtigt." → Realität

4. BEWEIS → "Das funktioniert wirklich." → Praxis
5. ENTSCHEIDUNG → "Ich will das." → Warum ich + Angebote + CTA

Die Website verkauft eine Methode. Aber Manuels stärkstes Argument ist ein Portfolio implementierter Praxis über 3 Institutionen (gibb, PH FHNW, UZH). Der Umbau stellt Praxis VOR Methode, ersetzt Prototyp-Tabs durch echte Cases, und macht die "Warum ich"-Sektion zum Beweis statt zur Biografie.
